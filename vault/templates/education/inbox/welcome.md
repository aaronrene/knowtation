# Welcome (students and instructors)

This template mirrors a real term: **syllabus → assignments → notes → schedule**.

## If you are a student

1. Copy the course slug into `project` on every note (`cs101-intro`).
2. After each lecture, file notes under `study-notes/` with **date + topic** in the title.
3. Track every graded item in `assignments/` even if the LMS also has a list—your vault can hold *your* scratch work and questions.

## If you are an instructor

- Maintain **one syllabus note** per course; link out to PDFs rather than duplicating policy prose unless you own the canonical copy.
- Use assignments notes for **rubric reminders** and common misconceptions to address next year.

## Import tip

Audio transcription imports belong in lecture notes: mark `source: transcription` in the body and clean up terminology in the same sitting, not later.

Remove this welcome note when the workflow sticks.

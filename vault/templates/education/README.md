# Education template

Use this layout for **courses, curriculum planning, study groups, and individual study** where notes must align with deadlines, readings, and recurring schedules.

## What you get

- **Inbox** — quick start for students and instructors.
- **Courses** — syllabus-style notes with weekly structure.
- **Assignments** — requirements, due dates, and status in one place.
- **Study notes** — lecture captures, including imports from transcription tools.
- **Resources** — curated links with short reviews.
- **Schedule** — time blocks that mirror how the term actually runs.

## Tips

- Keep **one project per course** in frontmatter (`cs101-intro`) so filters stay precise.
- When importing transcripts, add a **timestamped summary** at the top; keep raw transcript in an appendix or separate file if it is long.

## Frontmatter

Starter notes include `title`, `project`, `tags`, and `date` for consistent search and exports.

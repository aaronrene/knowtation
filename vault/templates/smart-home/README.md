# Smart home template

This template supports **home automation, IoT inventory, and household operations** in one place: devices, automations, energy, maintenance, and repeatable setup steps.

## What you get

- **Inbox** — onboarding and quick captures.
- **Devices** — one note per device or hub with specs and network details.
- **Automations** — human-readable descriptions of triggers, conditions, and actions.
- **Energy** — periodic usage logs (grid, solar, cost).
- **Maintenance** — service visits, checklists, and provider contacts.
- **Household** — SOPs guests and family can follow without asking you twice.

## Safety and privacy

- Store **IPs, passwords, and API keys** in a secrets manager or encrypted store; this template uses placeholders where needed.
- Treat guest Wi-Fi and camera notes as **sensitive**; restrict sync and backups accordingly.

## Frontmatter

Starter notes use `title`, `project`, `tags`, and `date` so they index cleanly beside the rest of your vault.

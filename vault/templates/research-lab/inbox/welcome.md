# Welcome to the lab knowledge base

This vault is where we keep **protocols, experiment logs, and decisions** so the whole team shares one source of truth. Email and chat are fine for coordination; durable knowledge lives here.

## Your first week

1. Read the latest **lab sync** note under `meetings/` and note any action items with your name.
2. Skim **active protocols** in `protocols/` that match your project—especially safety and waste handling.
3. Add yourself to the shared calendar and instrument booking system (links live in the lab wiki, not duplicated here).

## How we use tags

Use tags like `protocol`, `safety`, `instrument-qpcr`, or grant codes (`R01-xyz`) so searches stay fast. Prefer one primary `project` in frontmatter per note.

## Where to put new notes

- **Quick idea or link dump** → `inbox/`, then file weekly.
- **Paper you read** → `literature/` with citation metadata in the body.
- **A run you executed** → `experiments/` with date in the filename when possible.

If something feels missing, propose a short ADR under `decisions/` so we agree once and move on.

# Personal knowledge template

This template is for **individual knowledge workers** who want a small, opinionated vault layout without enterprise overhead.

## What you get

- **Inbox** — fast capture before you decide where something belongs.
- **Projects** — time-bound work with goals and milestones.
- **Areas** — ongoing responsibilities (health, finance, career) that never really “finish.”
- **Reference** — durable comparisons, specs, and decisions you will reopen.
- **Journal** — dated reflections tied to the same metadata as the rest of the vault.

## How to use it

1. Duplicate this folder into your vault (or symlink paths you like).
2. Keep **one project** field per note so search and filters stay honest.
3. Prefer short inbox notes; promote them into projects, areas, or reference when they mature.

## Frontmatter convention

Every starter note includes `title`, `project`, `tags`, and `date`. Adjust `project` to match how you name your own work.

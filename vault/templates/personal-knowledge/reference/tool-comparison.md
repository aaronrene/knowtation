# Note apps (Markdown-first)

Comparison for someone who wants **files on disk**, good search, and optional sync they control.

## Candidates

### App A — Folder of `.md` + desktop editor

- **Pros** — Zero lock-in; works with git; any editor.
- **Cons** — No built-in graph or backlinks unless you add plugins/scripts.

### App B — Dedicated knowledge app (local vault)

- **Pros** — Backlinks, daily notes, plugins; still local files.
- **Cons** — Heavier UX; some features tempt proprietary sync.

### App C — Hosted wiki

- **Pros** — Easy sharing; collaboration.
- **Cons** — Privacy and export story must be verified; offline weaker.

## Decision (for me)

Stay on **folder + indexer** for notes that must outlive any vendor. Use hosted wiki only for **public** docs.
